# @year/2000

A year 2000

## Install

~~~
npm install @year/2000
~~~

## Usage

~~~js
const day = require('@year/2000/01/01')
~~~

Returns a [Date](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date) object.
